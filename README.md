# Bomb-Notifier
A forge 1.12.2 mod for Champion ranked donators to use in Wynncraft to notify guild members of bombs.

# Known problems:
- Bombs thrown in same world will not trigger Bomb Bell, and hence will not have an alert button.

# Future additions:
